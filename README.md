# Rsbuild Project

## Setup

Instlar las dependencias

```bash
npm o pnpm install
```

## Get Started

Start the dev server:

```bash
npm o pnpm dev
```

Build the app for production:

```bash
npm o pnpm build
```

Preview the production build locally:

```bash
npm o pnpm preview
```
## Setting environment variables
**Required variables:**

* `API_URL`: API URL for the X service.

```bash
API_URL=http://localhost:4000
```
